# Legal Review Template

Framework for transportation attorney review before pilot installation. This is not legal advice — it's a structured set of questions for counsel.

## 1. Liability Assessment

### Raised Expectation / Duty to Maintain

**Question**: Once the coordinated illumination system is installed and operational, does the installing agency (TDOT, Nashville NDOT) assume a heightened duty to maintain the system in working condition? If the system fails to activate during a pedestrian crash, does the failure create greater liability than if the system had never been installed?

**Context for counsel**:
- System includes failsafe (mechanical timer relay provides basic illumination on push-button even if controller fails)
- System logs all activation events with timestamps — provides evidence of whether system was functional at time of any incident
- TDOT Traffic Operations Memo 2022 already mandates lighting at midblock crossings ("shall") — the question is whether the agency's liability is greater for installing a specific system that fails vs. not installing any coordinated system at all
- Tennessee Governmental Tort Liability Act (TCA 29-20-101 et seq.) provides sovereign immunity for discretionary design decisions

**Specific questions for counsel**:
- [ ] Is the decision to install coordinated illumination a discretionary design decision protected by sovereign immunity?
- [ ] Does the existing "shall" mandate for midblock crossing lighting create greater liability exposure for NOT installing than for installing a system that could fail?
- [ ] Does the failsafe design (degraded operation rather than complete failure) adequately address the duty-to-maintain concern?
- [ ] Does the data logging (timestamped activation records) create a discovery risk or a protective evidentiary record?

### Precedent

- [ ] Are there Tennessee or 6th Circuit cases addressing liability for traffic safety devices that were installed and subsequently failed?
- [ ] How has sovereign immunity been applied in Tennessee for traffic signal or lighting failures?
- [ ] Has any Tennessee court found enhanced liability for an agency that installed a safety improvement and failed to maintain it, beyond what existed before installation?

## 2. Dark Sky Variance (Nashville)

**Question**: Can the system be installed at 4000K CCT despite Nashville's BL2020-535 cap of 3000K?

**Three potential paths for counsel to evaluate**:

1. **Jurisdictional**: TDOT installations on state ROW follow TDOT standards, not Metro Nashville zoning. Does BL2020-535 apply to TDOT infrastructure on state routes within Nashville?
   - [ ] Review ordinance applicability language — does it apply to all outdoor lighting in Nashville, or only to construction reviewed under Metro building codes?
   - [ ] Has Nashville enforced the dark sky ordinance against any TDOT installation?
   - [ ] Does TDOT's Utility Accommodation Policy create a separate jurisdictional framework?

2. **Variance via BL2022-1088**: The Board of Fire and Building Code Appeals can grant variances for "manifest injustice" or where strict enforcement contradicts "spirit and purposes" or public interest.
   - [ ] Is pedestrian safety a sufficient basis for a "public interest" variance?
   - [ ] What is the Board's track record on variances? How often granted?
   - [ ] What documentation is needed for a variance application?

3. **Exemption argument**: The ordinance exempts downtown districts and MNAA property. Is there a principled basis for arguing that safety-critical activated lighting (15-20 seconds per event, not continuous) should be similarly exempt?
   - [ ] Would pursuing a legislative amendment adding a safety infrastructure exemption be more appropriate than a variance?

## 3. Procurement

**Question**: Can the system spec name specific products (e.g., TAPCO SafeWalk) in publicly bid procurement?

**Context**: TCA 12-4-903 prohibits single-brand specs without allowing "or equal" alternatives.
- [ ] Confirm the performance-based specification approach (lumens, beam angle, CCT, mounting type) satisfies TCA 12-4-903
- [ ] Review "or equal" determination process — who decides whether a proposed alternative meets the performance spec?
- [ ] If the pilot is funded as a research project (not a public works contract), does TCA 12-4-903 apply?

## 4. University Partnership

**Question**: Can a university fund installation of crosswalk infrastructure on public ROW?

**Potential structures**:
- [ ] Direct donation to TDOT/Nashville earmarked for specific crossing
- [ ] Cooperative agreement — university funds hardware, public agency installs and maintains
- [ ] Grant co-funding — university provides local match for federal safety grant
- [ ] Research partnership — university funds as research project, installation under public agency permit

**For counsel**:
- [ ] Which structure creates the cleanest liability boundary?
- [ ] Does the university assume any liability for the system's performance on public ROW?
- [ ] How is maintenance responsibility allocated in a cooperative agreement?
- [ ] Are there Tennessee precedents for university-funded public ROW improvements?

## 5. Right-of-Way and Permitting

- [ ] Confirm TDOT Use & Occupancy Permit process for solar equipment on state ROW
- [ ] Confirm Nashville encroachment permit requirements for city roads
- [ ] If the installation is a research project, does the permitting process differ?
- [ ] Who holds the permit — the university, the public agency, or the installing contractor?

## 6. Data and Privacy

- [ ] Does radar-based pedestrian detection constitute surveillance under any Tennessee or federal law?
- [ ] Do timestamped activation records create public records subject to FOIA/Tennessee Public Records Act (TCA 10-7-503)?
- [ ] If video data is collected for the evaluation study, what are the retention and access requirements?
- [ ] Does near-miss data (vehicles that don't yield) create an obligation to report or act on identified dangerous drivers?
